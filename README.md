# NavPLMun

NavPLMun is a mobile application designed to help users easily navigate the school premises of PLMun. The application provides location guidance and essential information about specific areas within the campus, helping students, visitors, and staff find their way efficiently.

This project exists to address the difficulty of navigating unfamiliar school locations by offering a simple, interactive, and accessible navigation solution through a mobile application.
